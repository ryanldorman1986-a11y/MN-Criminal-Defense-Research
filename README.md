# MN Criminal Defense Research Database

A comprehensive SQLite database for Minnesota criminal defense research.

## Quick Start

1. Initialize database: `python src/init_database.py`
2. Add statutes: `python src/add_mn_statutes.py`
3. Run menu: `python src/menu.py`

## Database Tables

- cases: Criminal cases
- statutes: Minnesota statutes (MN 609)
- precedents: Legal precedents
- defense_arguments: Defense strategies
- hearings: Hearing information
